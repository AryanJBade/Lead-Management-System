# Lead Management System

## Project Overview

Lead Management System is a full-stack web application developed using React.js, Node.js, Express.js, and MySQL.

The system allows managers and agents to manage leads efficiently through a secure dashboard with authentication and role-based access.

---

## Features

- JWT Authentication
- Role Based Access Control
- Dashboard Analytics
- Create Lead
- Update Lead
- Delete Lead
- Search Lead
- Lead Details View
- Auto Agent Assignment
- Activity Logs
- Protected Routes

---

## Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- Bootstrap

### Backend
- Node.js
- Express.js
- JWT
- BcryptJS

### Database
- MySQL
